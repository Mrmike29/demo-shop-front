# Project Installation

Download project.
### `git clone https://github.com/Mrmike29/demo-shop-front.git`

Update URL constant to your localhost.

Execute following commands.
### `npm i`
### `npm run build`
### `npm start`


# Project Functions (Backend required)
As Common and Admin User
* Sign Up 
* Log in
* Consult Products
* Add Product to Cart
* Consult your Cart
* Delete Product from Cart
* Order Products
* Consult your Orders
* Consult Companies
* Consult Inventory
* Download Inventory PDF
* Send PDF via Email
* Sign Out

Only for Admin User
* Create Products
* Delete Products
* Consult all Orders
* Create Company
* Delete Company
